# 🏢 Workspace Staff Manager

Application web interactive permettant de gérer la répartition des employés dans différents espaces d’un bâtiment, avec respect des règles métier selon les rôles.

---

## 🎯 Objectif

Ce projet a pour objectif de :
- Gérer l’affectation des employés dans différentes zones
- Appliquer des règles métier selon les rôles
- Offrir une interface intuitive et responsive
- Centraliser la gestion des employés et leur visualisation

---

## 🚀 Fonctionnalités implémentées

✅ Ajout d’un employé via un formulaire (nom, rôle, email, téléphone...)  
✅ Liste des employés non assignés (Unassigned Staff)  
✅ Affectation des employés à différentes zones  
✅ Suppression d’un employé d’une zone (retour à "Unassigned")  
✅ Affichage des zones :
- Salle de conférence  
- Réception  
- Salle des serveurs  
- Salle de sécurité  
- Salle du personnel  
- Salle d’archives  

✅ Application de certaines règles métier :
- Réception → Réceptionnistes uniquement  
- Salle des serveurs → Techniciens IT  
- Salle de sécurité → Agents de sécurité  

✅ Interface responsive (Desktop / Tablet / Mobile)  
✅ UI moderne avec Tailwind / CSS
